# JSer.info-editor

[JSer.infoアーカイブ](http://jser.info/jser.info/) のエディタアプリです。

- node-webkitアプリ

## Installation

```
git clone https://github.com/jser/jser.info.git /path/to/jser.info
```

起動したら、"設定"から`/path/to/jser.info`のパスを指定します。

リロードすると、一番新しいjser.infoのデータJSONが読み込まれます。
